# AP_DesenvolvimentoWeb
Atividade Prática da Disciplina de Ferramentas de Desenvolvimento Web
Este é um projeto desenvolvido para a atividade prática da Disciplina de Ferramentas de Desenvolvimento Web, utilizando HTML, CSS e JavaScript.
O objetivo foi praticar conceitos básicos de desenvolvimento web e organização de páginas.

1) Páginas do projeto: 
	•	Sobre mim
	•	Formação
	•	Portfólio
	•	Contato

2) Funcionalidades: 
	•	Navegação entre páginas por meio de links
	•	Layout utilizando HTML e CSS
	•	Formulário de contato com validação em JavaScript
	•	Estrutura organizada com header, conteúdo e footer

3) Tecnologias utilizadas: 
	•	HTML
	•	CSS
	•	JavaScript

4) Como visualizar: 
Você pode acessar o projeto pelo GitHub Pages ou baixar os arquivos e abrir o index.html no navegador.

5) Observação: 
Este projeto foi desenvolvido para fins de estudo e prática.
